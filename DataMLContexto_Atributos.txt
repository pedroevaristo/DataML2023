Atributos e seus significados

Dataset retirado de :
https://www.kaggle.com/datasets/rajkumarpandey02/bank-marketing-dataset

"The data is related with direct marketing campaigns of a Portuguese banking institution. 
The marketing campaigns were based on phone calls. Often, more than one contact to the same client was required, 
in order to access if the product (bank term deposit) would be ('yes') or not ('no') subscribed."

Foi utilizado do "bank.csv" que conta como uma amostra da dataset total "bank-full.csv".



X1 age: Idade (Por Ano) do cliente

X2 job:Tipo de emprego: 

                     Desconhecido = 1; Desempregado = 2; Técnico = 3; 
                     Estudante = 4; Services = 5; Autônomo = 6;
                     Aposentado = 7; Gerencia = 8; Doméstico = 9; Investidor = 10; 
                     Colarinho azul = 11; Administrador = 12;

X3 marital: Estado Civil: Solteiro = 1; Casado = 2; Divorciado = 3;

X4 education: Educação: Primario = 1; Secundário = 2; Terciario = 3; Desconhecido = 4;

X5 default: Indicates if the client has credit in default. :Não = 0; Sim = 1;

X6 balance: Average yearly balance of the client, in euros.

X7 Housing: Indicates if the client has a housing loan. :Não = 0; Sim = 1;

X8 loan: Indicates if the client has a personal loan. :Não = 0; Sim = 1;

X9 contact:Type of communication mode used for contact. : Desconhecido = 0; Telefone = 1; Celular = 2;

X10 day: Day of the last contact.

X11 campaign: Number of contacts performed during this campaign.

X12 pdays: Days since the client was last contacted from a previous campaign (999 indicates the client was not contacted).

X13 previous: Number of contacts before this campaign.

X14 poutcome: Outcome of the previous marketing campaign.:

Desconhecido = 0; Outro = 1; Sucesso = 2;  Falha = 3;

X15 y: Indicates if the client subscribed to a term deposit. :Não = 0; Sim = 1;
